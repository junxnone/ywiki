# 起名

  - 小名可以尽早取，胎教时，多喊喊小名
  - 出生证明是第一个需要大名的证件

## 小名

  - 生活中的事务命名(食物/节气/.....)
  - 记录宝宝宝妈的生活

## 大名

  - 问问长辈有什么讲究
  - 想要赋予宝宝什么寓意
